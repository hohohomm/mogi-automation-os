# Client Intake Form

Use this as a Tally form or Google Form. Every client fills this before work starts.

## Question List

### 1. Contact Info
- Name (text, required)
- Email (email, required)
- Phone (text, optional)
- Best way to reach you (select: Email / Phone / Platform DM)

### 2. Project Scope
- What do you need done? (paragraph, required)
  > *Be as specific as possible. What files, tools, or systems are involved?*

- What format is your input data in? (select all that apply)
  - CSV / Excel
  - PDF
  - Google Sheet
  - Database (SQL, Airtable, etc.)
  - Text files
  - Images / scanned documents
  - Other: _____

- What format should the output be? (select all that apply)
  - Clean CSV / Excel
  - Google Sheet
  - PDF report
  - JSON / API-ready data
  - Automated workflow
  - Dashboard
  - Other: _____

### 3. Current Process
- How are you doing this now? (text)
- How long does it take per week? (text)
- What's the hardest part? (text)

### 4. Tech Details
- Do you have sample files I can test with? (yes/no/optional)
  - *If yes, you'll be prompted to upload after submission*
- Are you using any specific tools/platforms? (text)
  - *E.g., Google Sheets, Xero, Shopify, WordPress, custom software*
- Do you have API access if needed? (yes/no/unsure)

### 5. Timeline & Budget
- When do you need this by? (select)
  - ASAP (this week)
  - Within 2 weeks
  - Within a month
  - No rush
- What's your budget range? (select)
  - Under $50
  - $50–$100
  - $100–$250
  - $250–$500
  - $500+
  - Not sure / Tell me your price
- Is this a one-time task or ongoing? (select: One-time / Ongoing / Not sure yet)

### 6. Confirmation
- Anything else I should know? (paragraph, optional)
- I confirm the information above is accurate (checkbox, required)
