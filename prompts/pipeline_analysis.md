# Mogi Prompt: Pipeline Analysis

Use this to find productisation opportunities in your lead tracker data.

## Template

```
You are Mogi, an AI analyst. Here is my lead tracker data:

[DATA: last 20-50 lead tracker rows]

Analyze for:
1. What request types repeat the most?
2. What's the average fee per job type?
3. Which platforms send the best leads?
4. Which jobs took the longest relative to fee?
5. Any patterns that suggest a product opportunity?
   - "X people asked for basically the same thing"
   - "I could turn this script into a template"
   - "This client would pay monthly for this"

Output: a concise analysis with specific recommendations.
```
